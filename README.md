# ExpressJS-Learning-Authentication

This is a simple application that demonstrates how to implement authentication in ExpressJS.

## Prerequisites

- NodeJS
- MySQL
- npm
- dotenv
- ejs
- express
- express-session
- cookie-parser
- bcrypt
- mysql2

## Installation

```bash
npm install // Install all the dependencies
```

## How to run

```bash
npm start // Start the server
```

## URL to open

`http://localhost:3000`
